# French Number to Words

A lightweight JavaScript library that converts numbers into their French textual representation.

Designed for financial, accounting, payroll, invoicing, ERP, and administrative applications where numeric amounts must also be written in words.

---

## Features

✔ Supports integers

✔ Supports large numbers

✔ French grammar rules

✔ Lightweight

✔ Easy integration

✔ Pure JavaScript

---

## Example

12345

↓

douze mille trois cent quarante-cinq
